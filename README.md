# YouStagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request secret
- [x] Confirm Secret
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [ ] Log in
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
